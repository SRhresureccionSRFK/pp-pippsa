SRFK++ Framework Flow
=====================

Client / Browser
       │
       ▼
  .htaccess / Front Controller
       │
       ▼
     index.php
       │
       ├─> core/functions.php  (autoload, helpers: pr, vr, prx, vrx, Console::log)
       │
       ├─> core/register.php   (Register::$config, constantes globales)
       │
       └─> core/boot.php       (DB::runDB, controller/action, excepciones)
               │
               ▼
         core/filters.php      (interceptors / middlewares)
               │
               ▼
         core/request.php      (GET/POST, friendly URLs, route mapping)
               │
               ▼
   app/controllers/ControllerBase
               │
               └─> app/controllers/PizzasController.php
                        │
                        ▼
               Models (app/models)
               │ - CRUD + magic methods
               │
               ▼
               Response
               │ - View (phtml)
               │ - JSON / AJAX / Redirect
               ▼
       Client / Browser
